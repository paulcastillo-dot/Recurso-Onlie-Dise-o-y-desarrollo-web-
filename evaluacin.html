Evaluación

Pregunta Verdadero-Falso

Pregunta 1

1. HTML es un lenguaje de programación que permite crear aplicaciones interactivas en la web.

Retroalimentación

Falso

HTML es un lenguaje de marcado, no de programación

Pregunta 2

2. Las páginas web estáticas tienen contenido fijo que solo cambia si el desarrollador lo modifica manualmente.

Retroalimentación

Verdadero

Las páginas estáticas muestran contenido que no cambia automáticamente; cualquier actualización requiere intervención manual del desarrollador.

Pregunta 3

3. Los hipervínculos externos enlazan a páginas o documentos dentro del mismo sitio web.

Retroalimentación

Falso

Los hipervínculos externos enlazan a sitios web fuera del sitio propio, mientras que los locales enlazan a páginas o secciones dentro del mismo sitio.

Pregunta 4

4. Los constructores visuales como WordPress permiten crear sitios web sin necesidad de conocimientos avanzados de programación.

Retroalimentación

Verdadero

Los constructores visuales facilitan la creación de sitios mediante plantillas y herramientas de arrastrar y soltar, ideales para principiantes.

Pregunta 5

5. HTML es importante porque establece la estructura de una página web, sobre la cual se aplican CSS y JavaScript.

Retroalimentación

Verdadero

HTML define la estructura de la página, mientras que CSS se encarga del diseño visual y JavaScript agrega interactividad y funcionalidad dinámica.
