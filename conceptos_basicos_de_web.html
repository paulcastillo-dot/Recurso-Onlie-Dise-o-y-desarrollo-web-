Conceptos basicos de Web

1. HTML
HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y organizar el contenido de las páginas web. Permite definir elementos como títulos, párrafos, listas, tablas, imágenes, enlaces y formularios.


Importancia: HTML es la base de cualquier sitio web, ya que establece la estructura sobre la cual se aplican estilos (CSS) y funcionalidades (JavaScript). Sin HTML, no sería posible mostrar contenido de forma organizada en un navegador.


Ejemplo: <h1>Mi primera página web</h1> crea un título principal en la página.


2. Páginas web

Páginas estáticas:

Contenido fijo que no cambia a menos que el desarrollador lo modifique manualmente.
Ideales para información institucional, portafolios o sitios de presentación de productos.
Ventaja: fácil de crear y rápido de cargar.
Desventaja: poco flexible para actualizar contenido constantemente.
Páginas dinámicas:

Contenido que puede cambiar automáticamente según la interacción del usuario, base de datos o información del servidor.
Ejemplos: foros, tiendas en línea, redes sociales.
Ventaja: permiten experiencias personalizadas y actualizaciones automáticas.
Desventaja: requieren más conocimientos de programación y recursos del servidor.

3. Hipervínculos


Locales:

Enlazan a documentos o secciones dentro del mismo sitio web.


Ejemplo: <a href="pagina2.html">Ir a la página 2</a>

Externos:

Enlazan a páginas web fuera del sitio web.


Ejemplo: <a href="https://www.google.com" target="_blank">Visitar Google</a>


Importancia: Los hipervínculos permiten la navegación entre diferentes páginas, mejorando la experiencia del usuario y conectando información de manera organizada.

4. Software de edición de sitios web

Editores de código:

Herramientas como VS Code, Sublime Text o Atom, donde el desarrollador escribe el código manualmente.
Ventajas: control total sobre el diseño, flexibilidad y soporte para múltiples lenguajes de programación.

Constructores visuales:

Plataformas como WordPress, Wix o Google Sites permiten crear páginas web de manera intuitiva, usando plantillas y herramientas de arrastrar y soltar.
Ventajas: no requieren conocimientos avanzados de programación, ideales para principiantes y creación rápida de sitios.
Importancia: Facilitan la creación y gestión de sitios web, adaptándose a diferentes niveles de conocimiento y necesidades del proyecto.
